The flow stroke

One continuous gradient stroke that rises, crosses and lifts — an exhale becoming an inhale. Violet melts into aqua; a coral node marks the turn. Built as a single path so it animates, embroiders or stamps cleanly.

Accent System — chakra & elementcore palette holds · accent flexes per class

A themed class adds one accent — a coloured stroke or node over the same flow mark — so each week reads as the same brand, retuned.

THE BLOOM BURST

A six-petal burst — half flower, half asterisk — punched into a black tile. Petals take any accent colour; the yellow centre is fixed. It works as a full lock-up or a tiny stamp at the end of a caption.

Accent System — chakra & elementthe accent takes the whole block

In Bloom the themed colour goes full-bleed — a complete colour-block takeover — while the type lock-up stays identical, so it's unmistakably the same brand at a glance.

Syne — Display / 800
Space Grotesk runs the system text — labels, schedules, captions — with total clarity. Upright throughout.
Space Grotesk — Body / 400–600

Brand Mark — original

The talisman seal

A geometric sun — ringed circle, solid core, four cardinal rays — reduced to pure flat shapes. No line-art fuss; just confident geometry that stamps, embroiders and scales to a favicon. The core is what re-colours for themed classes.

Accent System — chakra & elementthe seal core & rule re-colour

A themed class re-inks the seal's core and the section rule in the chakra or element colour. The geometry never changes — only the hue tells you what today holds.
